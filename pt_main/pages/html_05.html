input type="radio"

입력 태그 타입 radio는 라디오 버튼(or컨디션 (이거나 조건))

  • name을 같은 이름으로 여러 항목을 지정시 그룹화 된다.
  • label을 이용하여 해당 항목과 버튼을 연결해줄 필요가 있다.
  • value 값설정으로 전달할 값을 넣어 둘 수 있으나, 비워두어도 상관없다.
  • checked 를 이용하여 페이지가 로드 될때 초기 선택값을 넣어 둘 수 있다.
    cheackbox와는 달리 하나의 항목만 선택 가능하다. 


                            <label for="btn_radio1">radio1</label>
                            <input type="radio" name="radioGroup" id="btn_radio2">
                            <label for="btn_radio2">radio2</label>
                            <input type="radio" name="radioGroup" id="btn_radio3">
                            <label for="btn_radio3">radio3</label>
                            <input type="radio" name="radioGroup" id="btn_radio4">
                            <label for="btn_radio4">radio4</label>

input type="checkbox"

  • 입력태그타입 checkbox는 다중 선택을 지원 하는 input이며, label을 이용하여 해당 항목과 연결해야 한다.
  • value 값은 기존 input 과 동일
  • checked 옵션 역시 radio input 과 동일

                                <input type="checkbox" id="btn_chk1">
                                <label for="btn_chk1">check1</label>
                                <input type="checkbox" id="btn_chk2">
                                <label for="btn_chk2">check2</label>
                                <input type="checkbox" id="btn_chk3">
                                <label for="btn_chk3">check3</label>
                                <input type="checkbox" id="btn_chk4">
                                <label for="btn_chk4">check4</label>

<select>

  • select는 선택항목의 목록형 단축 메뉴 형식으로 만드는 입력 양식
  • 항목 수 만큼의 option이 필요하고, 이 안의 value로 값을 전달한다.
  • optgroup으로 중간 그룹을 설정 할 수 있다.
  • radio, checkbox처럼 외형 변경이 어렵다.

                                <label for="selectjob">직업 선택 :</label>
                                <select id="selectjob">
                                    <optgroup label="구직자">
                                        <option>무직</option>
                                        <option>학생</option>
                                    </optgroup>
                                    <optgroup label="근로자">
                                        <option>전문직</option>
                                        <option>공무원</option>
                                    </optgroup>
                                </select>

input type="image"

image 는 버튼을 웹사이트 디자인에 맞게 디자인된 버튼을 사용할 때 사용 한다.


                                <input type="image" src="../images/image_sample_ddu03.jpg" alt="imgButton">

input type="button"

button 은 최종 확인 전송 및 페이지 양식 되돌리기, 파일 찾아보기 기능을 제외한 나머지 기능의 버튼 (우편번호, 검색, 추가, 삭제 등)에 사용한다.


                                <input type="button" value="비밀번호확인">

input type="file"

file은 파일 찾아보기 버튼에 사용한다


                                <input type="file">

input type="file"

summit은 해당 폼 내 모든 input의 유효성을 검수하고 전송을 담당.
반드시 form 안에 위치해야함!!!!! 


                                <input type="submit" value="전송">

input type="reset"

reset은 해당 폼의 모든 양식 내용을 초기화(아예 없는 상태) 시킬 때 사용한다.
반드시 form 안에 위치해야함!!!!! 


                                <input type="reset" value="취소">

HTML5 Input's

input type="email"

type 을 email로 설정하면, 모바일에서 해당 필드에 focus 되었을 때, 키보드를 열면서 영문과 @버튼이 생긴다.
placeholder 로 예시로 적어둘 수 있다.(ie10이하 지원불가)

  • readonly : 읽기전용 (server에 저장이 된적이 있는 data를 불러오기) - 저장된적 있던 정보 - ex 아이디
  • disabled : 비활성화 (입력자체를 막아 server에 저장 자체가 불가능함.)
  • enabled : disabled의 counter - 활성화(입력가능 = 전송시 server저장가능)

                                <label for="userEmail">사용자 이메일 :</label>
                                <input type="email" id="userEmail" required readonly>
