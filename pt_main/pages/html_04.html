tabel 작성법에 대해 알아보자!

month / savings Table
Month Saving
January $100
February $80
Sum $180

<table> : tabel 을 만들때 쓰는 tag

  • <thead> : 테이블의 상단부(행) / 데이터의 분류 형식이 상단 분류일 경우 선택적 사용
  • <tbody> : 테이블의 실질적 데이터를 담당하는 부분 / 필수 사용
  • <tfoot> : 테이블의 하단부(행) / 데이터의 결산 형식이 필요할 경우 선택적 사용
    이렇게 삼총사가 기본 자식! 떄에따라서 thead와 tfoot은 생략하기도 함!
  • tr>th+td
    1. <tr> : table Row = 행
    2. <th> : table eHeading = 분류명 / cell(칸) = 열(column)과 동일
    3. <td> : td TableDate = 데이터 / cell(칸) = 열과 동일
  • <caption>: tabel 의 자식 tag, 테이블의 위의 제목을 담당한다.
  • <colgroup> : 열을 묶어서 스타일(css)지정 할 때 사용한다 > selector로 대체가 가능

                            <table>
                                <caption>month / savings Table</caption>
                                <colgroup>
                                   <col style="background-color: lightpink; width: 150px;">
                                   <col style="background-color: yellow; width: 150px;">
                                </colgroup>
                                <thead>
                                   <tr>
                                       <th>Month</th>
                                       <th>Saving</th>
                                   </tr>
                                </thead>
                                <tbody>
                                   <tr>
                                       <td>January</td>
                                       <td>$100</td>
                                   </tr>
                                   <tr>
                                       <td>February</td>
                                       <td>$80</td>
                                   </tr>
                                </tbody>
                                <tfoot>
                                   <tr>
                                       <th>Sum</th>
                                       <td>$180</td>
                                   </tr>
                                </tfoot>
                            </table>

2-1 3-1 2-2 2-3 3-3

<colspan> : 가로를 병합할 때 쓰는 태그 tag
<rowspan> : 세로를 병합할 때 쓰는 태그 tag

미세한 TIP! 우선 전부 표를 완성하고, 병합하는게 정신 건강에 이롭더라고....
그리고 필요없는행은 지우지말고 주석처리를 하는게 좋음 그게 보기 편함 ㅎ... 

                        <table>
                            <tbody>
                                <tr>
                                    <td colspan="3">1-1 1-2 1-3</td>
                                </tr>
                                <tr>
                                    <td rowspan="2">2-1 3-1</td>
                                    <td>2-2</td>
                                    <td rowspan="2">2-3 3-3</td>
                                </tr>
                                <tr>
                                    <!-- <td>3-1</td> -->
                                    <td>3-2</td>
                                    <!-- <td>3-3</td> -->   
                                </tr>
                            </tbody>
                        </table>
