hyperlink List에 대해 알아보자!

<hyperLink>를 이용하여 원하는 url 및 url 내 위치로 이동하는 법, 그외 기타 다양한 버튼 액션을 배워보자.
  • naver button
  • daum button
  • download button
  • email send button
  • cell button
  • named Anchor button

target 값

_self = 현재 시스템에서 세션을 유지해야 할 경우 ,
_blank = 세션을 유지할 필요가 없을 때 (아예 시스템이 바뀌는 상황) 새 탭으로 열어줘야함. 


                        <ul>
                            <li>
                                <a href="https://naver.com" target="_blank">naver button</a>
                            </li>
                            <li>
                                <a href="https://daum.net" target="_self">daum button</a>
                            </li>
                            <li>
                                <a href="../step2.zip">download button</a>
                            </li>
                            <li>
                                <a href="mailto:ysarang3000@gmail.com?subject='문의남깁니다.'">email send button</a>
                            </li>
                            <li>
                                <a href="tel:+821099559198">cell button</a>
                            </li>
                            <li>
                                <a href="#targetAnchor">named Anchor button</a>
                            </li>
                        </ul>

id와 a의 href 를 통일하면 이동할 수 있다!
보통 스크롤이 부드럽게 되는 css (혹은 style) 을 쓰거나 해서 많이 사용함. 


                        <style>
                            html,body{
                            margin: 0; height: 100%; scroll-behavior: smooth;
                            }
                        </style>
                

                    <ul id="menu">
                        <li><a href="#page1">1page</a></li>
                        <li><a href="#page2">2page</a></li>
